# Patente Nautica Trainer

Web app responsive per allenarsi ai quiz della patente nautica con modalità esame realistiche, allenamento continuo e studio con soluzioni.

## Funzionalità principali

- Flusso semplice in 2 passaggi: scelta modalità, poi configurazione.
- Modalità disponibili:
  - Esami e integrazioni
  - Quiz infinito personalizzato
  - Domande più sbagliate
  - Studio completo con soluzioni
- Simulazioni con soglie di superamento per sezione e riepilogo finale.
- Segnalazione errori direttamente dall'app con commento.
- Classifica domande più sbagliate basata sullo storico locale.
- Tema chiaro/scuro.
- Supporto immagini e layout ottimizzato desktop/mobile.
- Scorciatoie tastiera desktop:
  - `1..9` selezione risposta
  - `Enter` / `→` invia-avanti
  - `←` indietro
  - `E` spiegazione
  - `S` mostra soluzione

## Avvio locale

```bash
python3 -m http.server
```

Poi apri `http://localhost:8000`.
